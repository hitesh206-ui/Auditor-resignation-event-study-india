# Notebooks

Planned notebook workflow:

1. `01_event_data_cleaning.ipynb` - clean and validate auditor resignation event data.
2. `02_price_data_collection.ipynb` - collect stock and market index price data.
3. `03_event_study_calculation.ipynb` - estimate market models, abnormal returns, and CARs.
4. `04_summary_statistics.ipynb` - generate descriptive statistics and group summaries.
5. `05_regression_analysis.ipynb` - run cross-sectional CAR regressions.
6. `06_charts.ipynb` - generate publication-ready charts.

Notebook files will be added after the pilot dataset is collected.
